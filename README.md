# Microtech Stampings - React UI Clone

A pixel-perfect clone of the Microtech Stampings website built with React and plain CSS.

## Quick Start

### Prerequisites
- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation & Running

1. Navigate to the project directory:
```bash
cd microtech-stampings
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application will open automatically in your browser at `http://localhost:3000`.
